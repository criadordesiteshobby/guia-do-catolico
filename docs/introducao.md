# CATÓLICH CODE – Introdução

O CATÓLICH CODE é um projeto que une **programação**, **tecnologia** e **evangelização**, com o objetivo de criar um ambiente digital que ajude jovens católicos a aprender, crescer e se aprofundar na fé.

Este repositório contém:
- Arquivos do site
- Códigos fontes
- Documentação completa
- Materiais de evangelização

## Objetivos do Projeto
- Criar uma plataforma educativa baseada em programação e fé.
- Ensinar evangelização usando recursos digitais.
- Desenvolver ferramentas simples que possam ser utilizadas por qualquer pessoa.
- Ajudar iniciantes em programação a criar algo com propósito.
